LOTUS: LLMs Over Tables of Unstructured and Structured data

Enabling Semantic Queries with LLMs Over Tables of Unstructured and Structured Data

Discord Github Paper

Key Insights

The Lotus framework is a sophisticated method that integrates structured and unstructured data through a unique set of semantic operators. These operators, including semantic filtering, entity recognition, and cross-domain alignment, allow for advanced contextual querying, enhancing the interpretability of data relationships.

Semantic Operators

The key operators in Lotus include:

  • Semantic Filters: These allow filtering based on the content and structure of data. Lotus can interpret linguistic nuances within text and align these with structured data entries, enabling dynamic filtering based on research questions.
  • Entity Matching and Cross-Alignment: Entities within unstructured text are mapped to corresponding entities in structured data, leveraging knowledge graph alignment. This operator enhances query precision across mixed data sources.
  • Cross-Modal Embeddings: Multi-modal embeddings provide semantic understanding between text and table data, supporting queries that span both modalities. This is critical in applications that require context-sensitive retrievals from complex datasets.

Applications of Lotus

Lotus has wide applications in fields requiring deep data retrieval and analysis across heterogeneous datasets. Key sectors include:

  • Biomedical Research: Lotus enables semantic search across vast volumes of clinical trials, medical records, and research publications by linking entities such as drugs, genes, and diseases across different data formats.
  • Financial Analytics: Financial institutions can utilize Lotus for querying structured financial statements alongside unstructured news data to detect trends and risks, enabling timely decision-making.
  • Legal Text Processing: By integrating case law, legal contracts, and tabular legal data, Lotus supports efficient legal research, identifying relevant cases and statutes from diverse data sources.

Methodology

At the core of Lotus is a methodology that integrates multi-modal embeddings to enhance query flexibility. The framework enables natural language queries over structured data (tables) by leveraging language models trained for both numerical and textual reasoning. This architecture facilitates the interpretation of mixed data types, ensuring that domain-specific queries yield precise and actionable results.
